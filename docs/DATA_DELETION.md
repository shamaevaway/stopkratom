# Data deletion instructions

## Local data

Open Tea Taper → Profile → Data → Delete local data.

## Cloud data

Open Tea Taper → Profile → Account and cloud → Delete cloud data.

## Manual support deletion

Before public release, replace this with a real support email and process:

1. User sends account email.
2. Support verifies ownership.
3. Cloud data under `users/{uid}` is deleted.
4. Confirmation is sent.

If account creation is available in Google Play or App Store builds, a working account deletion path is required.
