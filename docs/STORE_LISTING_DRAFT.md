# Store listing draft

## Name

Tea Taper — Intake & Wellbeing Tracker

## Short description

Private tracker for reducing tea intake and monitoring daily wellbeing.

## Full description

Tea Taper is a private journaling app for people who want to track kratom tea intake, daily goals, triggers, and wellbeing while reducing consumption over time.

Features:

- Record intake in milliliters
- Edit entries by date and time
- Track daily targets and progress
- Monitor 15 wellbeing metrics from 0 to 10
- Log triggers and helpful actions
- Review weekly progress
- Export CSV/JSON
- Optional cloud sync
- Offline-first design

Tea Taper is not a medical device and does not provide medical treatment, diagnosis, or advice. If you need medical help, contact a qualified professional.
