# Kratom Log

Локальное PWA-приложение для iPhone/Android/desktop: фиксация объема чая кратома в мл, быстрые добавления, дневные суммы, график, цель, экспорт/импорт.

## Что есть

- Добавление приемов по 50/100/150/200/250/300 мл или вручную.
- Автоматическая сумма за день.
- График за 7/14/30/90 дней.
- Дневная цель и предупреждающий порог.
- История сегодняшних записей и последние 30 дней.
- Экспорт JSON, импорт JSON, экспорт CSV.
- Offline-first через service worker.
- Данные хранятся локально в браузере устройства.

## Как поставить на iPhone

Вариант без App Store:

1. Залить папку на любой статический HTTPS-хостинг: GitHub Pages, Netlify, Vercel, Cloudflare Pages или свой домен.
2. Открыть ссылку в Safari на iPhone.
3. Нажать «Поделиться» → «На экран Домой».
4. Если iOS показывает переключатель, включить «Открывать как веб-приложение».
5. Открывать приложение с иконки на главном экране.

Важно: данные локальные, но сама оболочка должна открываться по ссылке. После первого открытия приложение кэшируется и работает офлайн.

## Как быстро проверить на компьютере

```bash
cd kratom-tracker-pwa
python3 -m http.server 8080
```

Открыть `http://localhost:8080`.
